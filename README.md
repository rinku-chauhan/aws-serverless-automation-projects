# AWS Serverless Automation Projects

A collection of hands-on AWS Serverless Automation projects built using AWS Lambda, Python, Boto3, EC2, S3, IAM, and EBS.

### EC2 Automation
Automated EC2 instance start and stop operations based on tags.

### S3 Cleanup
Automatically delete S3 objects older than 30 days.

### S3 Encryption Monitor
Detect S3 buckets without server-side encryption enabled.

### EBS Snapshot Automation
Create EBS snapshots and delete snapshots older than 30 days.

## Technologies Used

- AWS Lambda
- Amazon EC2
- Amazon S3
- Amazon EBS
- IAM
- Python
- Boto3
