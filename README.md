# AWS Serverless Automation Projects

A collection of hands-on AWS serverless automation projects built using **AWS Lambda**, **Python**, **Boto3**, **EC2**, **S3**, **IAM**, and **EBS**.

This repository demonstrates how cloud operations can be automated using serverless architecture to improve efficiency, security, and infrastructure management.

## Projects Included

### 1. EC2 Instance Automation

Automatically start and stop EC2 instances based on resource tags.

### 2. S3 Bucket Cleanup Automation

Automatically removes objects older than 30 days from an S3 bucket.

### 3. S3 Encryption Monitoring

Monitors S3 buckets and reports encryption status.

### 4. EBS Snapshot Automation

Automates EBS snapshot creation and retention management.

## Technologies Used

- AWS Lambda
- Amazon EC2
- Amazon S3
- Amazon EBS
- IAM
- Python
- Boto3
